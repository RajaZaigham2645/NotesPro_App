# NotesPro App

NotesPro is a modern Android note-taking application built using Java, XML, and Firebase Authentication. The app allows users to securely create, edit, delete, and manage their notes with a clean and user-friendly interface.

## 📱 Features

- 🔐 Firebase Authentication (Login & Sign Up)
- 📝 Create Notes
- ✏️ Edit Notes
- 🗑️ Delete Notes
- 🔍 Search Notes Instantly
- 🌙 Dark Mode Support
- ☀️ Light Mode Support
- 📋 View App Version
- 🚪 Secure Logout
- 🎨 Clean and Responsive User Interface

## 🛠️ Technologies Used

- Java
- XML
- Android Studio
- Firebase Authentication
- Material Design Components

## 🚀 Installation

1. Clone the repository:

```bash
git clone https://github.com/RajaZaigham2645/NotesPro_App.git
```

2. Open the project in Android Studio.

3. Connect your Firebase project.

4. Add your `google-services.json` file to the `app` directory.

5. Sync Gradle and run the application.
